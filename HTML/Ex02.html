Basic Html Tags

Every HTML document has the following things:

Headings: Used to provide the headings to the document with tags from <h1>(Biggest) to <h6>(Smallest).

Sections: Blocks that has replaced divs within the html document. They are created using <section> tag.

Paragraphs: Similar to divs but used to present small amount of textual content within the div. Provided wth a tag called <p>. p cannot be nested but divs can be nested.

Spans: Inline Content are called as Spans. Used with a tag called <span>.
