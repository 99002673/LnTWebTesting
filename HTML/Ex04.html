Lists in HTML

Lists in HTML docs are of 2 types: Ordered and UnOrdered.

Orderd lists

listing is based on numbering where the number type could be alphabetic or numeric. For Eg, list of trainings currently in C-TEA, Mysore

  1. C# Training
  2. Spring Training
  3. Cloud Computing
  4. JavaScript and Jasmine
  5. Java Programming

Unordered lists are bulletted lists where the content is of no specific order but a simple listings. Typical would be ToDo list that U see in UR Mobile Apps

  • Visit a Temple
  • Shop for the groceries
  • Buy a Pair of Shoes
  • Visit a Dentist
  • Pack my Bags
